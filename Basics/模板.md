## 1. 函数模板

函数模板是通用的函数描述，也就是可以用泛型来定义函数，其中泛型可以用具体的类型（如int 或 double）替换。  
函数模板允许以仍以类型的方式来定义函数。例：
``` C++
template <typename AnyType>
void Swap(AnyType &a, AnyType &b)
{
	AnyType temp;
	temp = a;
	a = b;
	b =temp;
}

```
> *template*和*typename*是关键字
