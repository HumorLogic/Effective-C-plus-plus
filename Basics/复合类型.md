## 3. 结构体Struct

**结构体**是一种用户自定义的可用的数据类型，它可以存储不同类型的数据项，关键字是<code><b>struct</b></code>。  

### 3.1 声明一个结构体
例如某公司要创建一种类型来描述其生产线上充气产品的成员。这种类型应存储产品名称、容量和售价。下面创建相应的结构体：  
```C++
struct inflatable     //结构体声明
{
    char name[20];
    float volume;
    double price;
}
```
