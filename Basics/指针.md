## 1.指针介绍  
**指针** 是一个变量，用来存储的是值的地址，而不是值本身。通过指针，可以简化一些 C++ 编程任务的执行，  
还有一些任务，如动态内存分配，没有指针是无法执行的。

## 2.指针声明  
格式：  
`typeName * pointerName`;  
示例：  
``` C++
double * pn;  //pn可以指向一个双浮点值
char * pc;    //pc可以指向一个字符型值
```  
`pn`和`pc`都是指针，而`double*`是指向`double`类型的指针，`char*`是指向`chare`类型的指针，
