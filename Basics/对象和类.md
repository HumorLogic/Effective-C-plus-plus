## 2.0 创建一个简单的类
Rectangle类：
``` C++
// Rectangle.h
#ifndef RECTANGLE_H
#define RECTANGLE_H

#include <iostream>

class Rectangle		// 类定义
{
public:
	float pos_x;
	float pos_y;
	void Draw();
	void SetSize(int w,int h);

private:
	int width;
	int height;

};


#endif // !RECTANGLE_H
```
