## 5. 函数指针
### 5.1 获取函数的地址
获取函数的地址很简单，只要使用函数名即可（后面不跟参数）。
``` C++
void Print(string str){
   cout<<str;
}

void main(){

cout<<Print<<endl;      // 打印出Print()函数地址

}
```
